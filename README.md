# Sobre o projeto

- Este projeto não é exatamente um projeto, inicialmente, era para ser apenas testes sobre PencilKit, como angulação e pressão da ApplePencil, diferentes ferramentas, como persistir desenhos e etc, mas, no fim esse "aplicativo" apenas com um canva se tornou algo essencial para o desenvolvimento de [SKEDIA](https://apps.apple.com/br/app/skedia/id6745489564), Quinto Challange Apple Developer Academy | Senac turma 2024 - 2025
- Ele foi utilizado para coletar rabiscos aleatórios de pessoas pela universidade, armazenar, e converter em metadados, para que os rabiscos possam ser utilizados no aplicativo real
- Feito para IPadOS

# Como executar o jogo

- Você não vai querer executar isso, mas
- Você vai precisar de um iPad e um mac, clone o projeto no XCode e execute no iPad
